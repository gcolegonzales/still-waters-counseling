# Add Laura's photo

The About section shows a provider card for **Laura Gonzales, LMSW, LCSW**.
Until a real photo is added, it shows a tasteful "LG" monogram placeholder.

To add her photo:

1. Save the portrait she sends as **`laura-gonzales.jpg`** in this `assets/` folder
   (this exact filename — lowercase, hyphenated).
2. Best results: a roughly **portrait / vertical** photo (4:5), head-and-shoulders,
   at least ~800px wide. It will be cropped to fill the card.
3. Commit + push — it appears automatically (the card is wired to
   `assets/laura-gonzales.jpg`, and the monogram hides once the file loads).

No code changes needed.
